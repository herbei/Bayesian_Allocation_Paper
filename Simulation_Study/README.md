# Reproducibility Code

This folder contains the code prepared for submission with the Spatial Statistics manuscript.

- `Simulation_Study/` contains the code and compact inputs for Section 4.
- `SST_Application/` contains the code and compact processed inputs for Section 5.

The two analyses are intentionally separated. Generated MCMC output, logs, rendered figures,
local package libraries, backups, and raw climate-model NetCDF files are not included.

Required R packages across both analyses are `ggplot2`, `gridExtra`, `maps`, `scales`,
`Matrix`, and `ncdf4`. Base R packages are not listed.
